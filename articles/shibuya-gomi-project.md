# 渋谷区の路上ゴミ問題の記事を読んだ

あれ、なかなか象徴的な記事ですよね。

渋谷区の路上ゴミ問題って、

* **路上飲み・深夜帯の行動**
* **「公共空間を誰が管理するのか」問題**

* ゴミ箱撤去（テロ対策・管理コスト）

* モール集約
* 動線管理
* 半公共空間の設計

の話と、実はかなり地続きです。

この件、

* 行政の失敗だと思いました？
* それとも「都市の性格上、不可避」だと思いました？

どっち寄りで読んだか、ちょっと気になります。

# 記事の方向的には、路上飲みの文化からの派生と見てるようだが

うん、その読み取りで合ってると思います。
あの記事、**かなり明確に「路上飲み文化 → 路上ゴミ」**という因果で描いてますね。

整理すると、記事の“方向性”はだいたいこんな感じです。

### 記事のフレーミング

* 渋谷では
  **路上飲みが常態化**
* ↓
  コンビニ・テイクアウトで飲食
* ↓
  **その場で消費 → その場にゴミが残る**
* ↓
  路上ゴミが可視化・常態化

つまり
**「飲み方の変化（文化）」が先にあり、ゴミはその副産物**
という描き方ですね。

### ここがミソ

この記事はあえて、

* モラルの欠如
* 観光客のマナー
  を前面には出していない。

代わりに、

* 路上飲みという「行動様式」
* 夜間・週末に集中する人流
* 店内消費から屋外消費へのシフト

を**都市文化の変化**として扱ってる。

これは朝日（AERA系）らしい視点でもあります。

### ただし、あなたが感じてそうな違和感

たぶんこれですよね👇

> 「じゃあ路上飲みをやめさせればいい」

  * 夜間インフラの空白
  * 半公共空間の不在

ここを突かずに「文化」に寄せると、
**設計の問題が倫理の話にすり替わる**。
